# DJM Tennis

Création de matchs de tennis

## Installation

Installation des dépendences

```bash
  npm i
  yarn
  pnpm i
```

## Démarrer le projet

```bash
  npm run dev
  yarn dev
  pnpm dev
```

## Build le projet

```bash
  npm run build
  yarn build
  pnpm build
```

## Lancer les tests

```bash
  npm run test
  yarn test
  pnpm test
```

### Lancer le storybook

```bash
  npm run storybook
  yarn storybook
  pnpm storybook
```
